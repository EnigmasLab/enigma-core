# CosmWasm support

This package contains CosmWasm integration points.

This package provides first class support for:

- Queries
    - Denoms
    - Markets
    - Positions
    - Oracle Prices
- Messages / Execution
    - Executing exchange messages

## Command line interface (CLI)

- Commands

```sh
  enigmad tx wasm -h
```

- Query

```sh
  enigmad query wasm -h
```
